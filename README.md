# Apostilas para cursos técnicos

Este repositório reúne materiais didáticos em HTML com foco em linguagem clara, organização visual e uso em sala de aula ou laboratório.

## Arquivos disponíveis

- `eletronica.html` — apostila de fundamentos da eletrônica para cursos técnicos.
- `desenvolvimentoWeb.html` — apostila de desenvolvimento web com HTML, CSS e JavaScript, exercícios, atividades e curadoria de estudos.

## Público-alvo

Estudantes de cursos técnicos, especialmente turmas de informática e áreas correlatas, que precisam de material introdutório, didático e aplicável a projetos práticos.

## Características dos materiais

- Arquivos HTML únicos, fáceis de abrir no navegador.
- CSS e JavaScript embutidos no próprio documento.
- Conteúdo organizado em módulos.
- Exercícios e atividades práticas integrados ao material.
- Elementos visuais como tabelas, blocos de código, figuras e destaques pedagógicos.
