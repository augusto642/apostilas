Arquivo: [`apostila_eletronica.html`](apostila_eletronica.html)

Apostila completa de nível técnico-profissional em português, cobrindo:

| Módulo | Conteúdo |
|--------|----------|
| 1 | Fundamentos: Eletricidade × Eletrônica × Eletrônica de Potência, história, estrutura atômica |
| 2 | Grandezas elétricas: Tensão, Corrente, Resistência, Potência, Lei de Ohm, Kirchhoff, Thévenin |
| 3 | Componentes passivos: Resistores (código de cores), Capacitores (RC), Indutores, Transformadores |

| 9 | Eletrônica de Potência: PWM, Ponte H, inversores solares |

| Apêndice | Glossário técnico, tabela de normas (ABNT/IEC), referências bibliográficas, ferramentas |

### Como usar

Abra o arquivo `apostila_eletronica.html` em qualquer navegador moderno. O arquivo é
totalmente auto-contido (sem dependências externas) e pode ser impresso ou salvo como PDF.

### Recursos incluídos

- ✅ Diagramas SVG interativos (estrutura atômica, circuitos, formas de onda PWM)
- ✅ Tabelas comparativas de componentes e normas técnicas
- ✅ Exemplos resolvidos com passo a passo
- ✅ Exercícios com respostas comentadas (ocultas/reveladas com clique)
- ✅ Código Arduino comentado (ADC, PWM, sensor NTC)
- ✅ Conformidade com ABNT NBR 5410, IEC 61131, IEC 61800
- ✅ Design responsivo — funciona em desktop, tablet e mobile
